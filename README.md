# Bulk STK Push Dashboard

## Features

- Bulk STK push requests
- 2000ms delay between each request
- Simple dashboard UI
- Render deployment ready
- GitHub ready

## Installation

```bash
npm install
```

## Configure Environment Variables

Rename `.env.example` to `.env` and add your credentials.

## Run Locally

```bash
node server.js
```

Open:

http://localhost:3000

## Deploy to Render

1. Push to GitHub
2. Login to Render
3. Create Web Service
4. Add environment variables
5. Deploy
